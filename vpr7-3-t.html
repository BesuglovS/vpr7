📚 Теория: Типы файлов и расширения

🔬 Что нужно знать о типах файлов?

В задании №3 ВПР по информатике нужно уметь сопоставлять расширения файлов с их типами. Расширение файла указывает на его назначение и определяет, какая программа откроет этот файл.

Ключевые понятия:

  • Расширение файла — это конечная часть имени файла, которая указывает на его тип (например, .docx, .jpg, .mp3)
  • Тип файла — категория, определяющая назначение файла: текстовый документ, изображение, аудиозапись и т.д.
  • Ассоциация — связь между расширением и программой, которая открывает такие файлы

📐 Структура имени файла

имя_файла.расширение

Примеры: документ.docx, фото.jpg, музыка.mp3

📄

Текстовые файлы

Текстовые файлы — содержат текстовую информацию: документы, статьи, отчёты, заметки. Могут включать форматирование, таблицы, изображения.
.txt Обычный текст
.doc Word 97-2003
.docx Word 2007+
.odt OpenDocument
.pdf PDF документ
.rtf Формат. текст

🖥️ Программы для открытия:

Microsoft Word LibreOffice Writer OpenOffice Writer Блокнот Adobe Reader (PDF)
📊

Презентации

Презентации — файлы для демонстрации слайдов с текстом, изображениями, графиками и мультимедиа. Используются для выступлений и обучения.
.ppt PowerPoint 97-2003
.pptx PowerPoint 2007+
.odp OpenDocument

🖥️ Программы для открытия:

Microsoft PowerPoint LibreOffice Impress OpenOffice Impress Google Slides
📈

Электронные таблицы

Электронные таблицы — файлы для работы с числовыми данными, расчётами, диаграммами. Содержат ячейки с формулами и функциями.
.xls Excel 97-2003
.xlsx Excel 2007+
.ods OpenDocument

🖥️ Программы для открытия:

Microsoft Excel LibreOffice Calc OpenOffice Calc Google Sheets
🖼️

Графические файлы (изображения)

Графические файлы — содержат изображения: фотографии, рисунки, схемы. Различаются по способу кодирования (растровые и векторные) и степени сжатия.
.jpg Фотографии
.png С прозрачностью
.gif Анимация
.bmp Без сжатия
.tiff Высокое качество
.svg Векторная графика

🖥️ Программы для открытия:

Просмотр фотографий Paint Adobe Photoshop GIMP Браузеры
🎵

Аудиофайлы

Аудиофайлы — содержат звуковую информацию: музыку, голосовые записи, звуковые эффекты. Различаются по качеству и степени сжатия.
.mp3 Популярный формат
.wav Без сжатия
.ogg Свободный формат
.aac Высокое качество
.flac Без потерь
.wma Windows Media

🖥️ Программы для открытия:

Windows Media Player VLC Winamp AIMP
🎬

Видеофайлы

Видеофайлы — содержат видеоизображение и звук: фильмы, клипы, записи с экрана. Различаются по кодекам и контейнерам.
.mp4 Универсальный
.avi Классический
.mkv Matroska
.mov QuickTime
.webm Для веба
.wmv Windows Media

🖥️ Программы для открытия:

VLC Windows Media Player Media Player Classic Браузеры
📦

Архивы

Архивы — сжатые файлы или группы файлов. Используются для экономии места и удобной передачи нескольких файлов одним пакетом.
.zip Универсальный
.rar WinRAR
.7z 7-Zip

🖥️ Программы для открытия:

7-Zip WinRAR WinZip Проводник Windows
🌐

Веб-страницы

Веб-страницы — файлы с кодом для отображения в браузере. Содержат HTML-разметку для структуры и отображения информации.
.html HTML страница
.htm Краткая форма

🖥️ Программы для открытия:

Браузеры (Chrome, Firefox, Edge) Текстовые редакторы Visual Studio Code
⚙️

Исполняемые файлы

Исполняемые файлы — программы, которые можно запустить на выполнение. Содержат машинный код, понятный компьютеру.
.exe Программа Windows

⚠️ Важно:

Исполняемые файлы могут содержать вирусы! Запускайте только те файлы, которые получены из надёжных источников.

Советы для запоминания

💡 Как запомнить расширения?

  • .mp3, .mp4 — MPEG формат (MPEG-1 Audio Layer 3, MPEG-4)
  • .doc, .docx — DOCument (документ)
  • .xls, .xlsx — eXceL Spreadsheet (таблица Excel)
  • .ppt, .pptx — PowerPoinT (презентация)
  • .jpg — JPEG (Joint Photographic Experts Group)
  • .png — Portable Network Graphics (переносимая сетевая графика)
  • .gif — Graphics Interchange Format
  • .html — HyperText Markup Language
  • .exe — EXEcutable (исполняемый)
  • .od* — OpenDocument (открытый стандарт: .odt — text, .ods — spreadsheet, .odp — presentation)

📝 Легко запомнить:

  • Аудио — .mp3 (самый популярный)
  • Видео — .mp4 (самый популярный)
  • Фото/рисунок — .jpg или .png
  • Текст — .docx или .txt
  • Таблица — .xlsx
  • Презентация — .pptx
  • Архив — .zip

⚠️ Частые ошибки

  • Путать .doc/.docx (документ) с .xls/.xlsx (таблица)
  • Путать .ppt/.pptx (презентация) с .pdf (документ)
  • Забывать про форматы OpenDocument: .odt, .ods, .odp
  • Путать аудио (.mp3) с видео (.mp4)
