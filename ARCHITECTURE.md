# Архитектура проекта vpr7

## Обзор

Статическое веб-приложение для подготовки к Всероссийской проверочной работе (ВПР) по информатике для 7 и 8 классов. Содержит интерактивные задания с drag-and-drop, режим экзамена и генератор вариантов для печати.

## Структура проекта

```
vpr7/
├── vpr.html                   # Главная (выбор класса: 7 или 8)
├── js/                        # Библиотеки
│   ├── jspdf.umd.min.js       # Генерация PDF
│   └── html2canvas.min.js     # HTML → Canvas
│
├── vpr7/                      # 7 класс (12 заданий)
│   ├── vpr7.html              # Выбор задания
│   ├── vpr7-exam.html         # Режим экзамена
│   ├── vpr7-generator.html    # Генератор вариантов
│   ├── vpr7-{1-12}.html       # Задания 1–12
│   ├── vpr7-{1-12}-t.html     # Теория к заданиям
│   ├── css/                   # Стили (общие + per-task)
│   ├── js/
│   │   ├── vpr7-{1-12}-generator.js  # Генераторы данных заданий
│   │   ├── vpr7-exam-utils.js        # Таймер, подсчёт баллов

├── vpr8/                      # 8 класс (10 заданий)
│   ├── vpr8.html              # Выбор задания
│   ├── vpr8-exam.html         # Режим экзамена
│   ├── vpr8-generator.html    # Генератор вариантов
│   ├── vpr8-{1-10}.html       # Задания 1–10
│   ├── vpr8-{1-10}-t.html     # Теория к заданиям
│   ├── css/                   # Стили
│   ├── js/                    # Генераторы и утилиты

├── metrika.js                 # Яндекс.Метрика
└── LICENSE                    # MIT
```

## Архитектура приложения

### Навигация

```
vpr.html (выбор класса)
├── vpr7/vpr7.html (выбор задания)
│   ├── vpr7-{1-12}.html      (12 заданий)
│   ├── vpr7-exam.html        (экзамен)
│   └── vpr7-generator.html   (генератор PDF)
│
└── vpr8/vpr8.html (выбор задания)
    ├── vpr8-{1-10}.html      (10 заданий)
    ├── vpr8-exam.html        (экзамен)
    └── vpr8-generator.html   (генератор PDF)
```

### Типы интерактива

| Технология | Назначение |
|------------|------------|
| Drag-and-drop API | Перетаскивание элементов в категории |
| Touch events | Поддержка сенсорных экранов |
| jsPDF | Генерация PDF-вариантов |
| html2canvas | Скриншоты страниц для PDF |

### Генераторы заданий

Каждое задание имеет свой генератор данных (`vpr7-{n}-generator.js`), который:

1. Загружает текстовые данные из `data/*.txt`
2. Случайным образом формирует варианты
3. Генерирует HTML-разметку задания

### Формат данных

Текстовые файлы в `data/` используют простую разметку:

```
@Название категории
Элемент 1
Элемент 2
Элемент 3
```

Генераторы парсят этот формат и создают задания с drag-and-drop интерфейсом.

### Режим экзамена

```
vpr7-exam.html / vpr8-exam.html
├── Таймер (vpr7-exam-utils.js)
├── Все задания подряд
├── Навигация между заданиями
├── Подсчёт баллов
└── Итоговая оценка
```

### Генератор вариантов

```
vpr7-generator.html / vpr8-generator.html
├── Выбор количества вариантов
├── Генерация уникальных вариантов
├── Рендер в html2canvas
└── Экспорт в PDF (jsPDF)
```

### Хранение данных

| Ключ | Назначение |
|------|-----------|
| `vpr7-progress` | Пройденные задания (7 класс) |
| `vpr8-progress` | Пройденные задания (8 класс) |
| `vpr7-theme` | Предпочтения темы |

## Дизайн-система

Общая с экосистемой:

- Фон: тёмный градиент
- Карточки: полупрозрачные с цветными акцентами
- Drag-and-drop: визуальная обратная связь при размещении
- Цветовая тема: синяя (соответствует ВПР)
