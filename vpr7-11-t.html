📚 Теория: Цветовая модель RGB

🌈 Что такое цветовая модель RGB?

RGB (Red, Green, Blue — красный, зелёный, синий) — аддитивная цветовая модель, описывающая способ кодирования цвета для вывода на экраны мониторов, телевизоров и других устройств.

Ключевые понятия:

  • Аддитивная модель — цвета складываются (смешиваются из света), в отличие от субтрактивной модели (краски на бумаге)
  • Три основных цвета — красный (R), зелёный (G) и синий (B)
  • Диапазон значений — каждый компонент принимает значение от 0 до 255
  • Всего цветов — 256³ = 16 777 216 различных оттенков
🔵

Основы цветовой модели RGB

RGB — это способ представления цвета в виде трёх чисел: интенсивности красного, зелёного и синего компонентов. Каждый компонент может принимать значение от 0 (минимум) до 255 (максимум).

📊 Как работает модель RGB

В модели RGB каждый цвет кодируется тремя числами в формате RGB(R, G, B), где:

  • R (Red) — интенсивность красного цвета (от 0 до 255)
  • G (Green) — интенсивность зелёного цвета (от 0 до 255)
  • B (Blue) — интенсивность синего цвета (от 0 до 255)

💡 Почему именно эти три цвета?

Красный, зелёный и синий выбраны потому, что они являются основными цветами света. Человеческий глаз имеет три типа рецепторов (колбочек), чувствительных именно к этим трём цветам. Смешивая их в разных пропорциях, можно получить любой видимый цвет!

🔢 Диапазон значений

Значение Смысл Пример
0 Компонент полностью выключен (отсутствует) R=0 — нет красного
255 Компонент на максимуме R=255 — максимальный красный
128 Половина интенсивности R=128 — тёмно-красный

⚠️ Важно помнить!

  • Чем больше значение, тем ярче компонент
  • Все нули (0, 0, 0) дают чёрный цвет (нет света)
  • Все максимумы (255, 255, 255) дают белый цвет (весь свет)
  • Это аддитивная модель — цвета складываются, а не вычитаются

Основные цвета в RGB

Знание основных цветов и их RGB-кодов поможет быстро определять цвет по значениям компонентов. Запомните эти базовые сочетания!

📋 Таблица основных цветов

Цвет R G B Образец Как запомнить
Белый 255 255 255 Все максимумы
Чёрный 0 0 0 Все нули
Красный 255 0 0 Только R
Зелёный 0 255 0 Только G
Синий 0 0 255 Только B
Жёлтый 255 255 0 Красный + Зелёный
Голубой (Циан) 0 255 255 Зелёный + Синий
Пурпурный (Маджента) 255 0 255 Красный + Синий
Серый 128 128 128 Все равны

🔄 Смешение основных цветов

Красный Зелёный Синий Жёлтый Голубой Пурпурный Белый

✅ Правила смешения цветов

  • Красный + Зелёный = Жёлтый (R=255, G=255, B=0)
  • Зелёный + Синий = Голубой (Циан) (R=0, G=255, B=255)
  • Красный + Синий = Пурпурный (Маджента) (R=255, G=0, B=255)
  • Красный + Зелёный + Синий = Белый (R=255, G=255, B=255)
  • Отсутствие всех цветов = Чёрный (R=0, G=0, B=0)

🌈 Оттенки серого

💡 Секрет серого цвета

Когда все три компонента равны (R = G = B), получается оттенок серого:

  • (0, 0, 0) — чёрный
  • (64, 64, 64) — тёмно-серый
  • (128, 128, 128) — серый
  • (192, 192, 192) — светло-серый
  • (255, 255, 255) — белый

🎯 Как определять цвет по значениям RGB

  1. Проверьте, равны ли все компоненты — если да, это оттенок серого
  2. Определите доминирующий компонент(ы) — какой имеет наибольшее значение
  3. Проверьте наличие двух высоких компонентов — например, R=255 и G=255 дают жёлтый
  4. Учитывайте интенсивность — низкие значения дают тёмные оттенки

📋 Быстрый анализ цвета

Условие Результат

Два компонента = 255, третий = 0 Смешанный цвет (жёлтый, голубой, пурпурный)
Все компоненты = 255 Белый
Все компоненты = 0 Чёрный

Примеры решения задач

Разбор типовых задач из ВПР по информатике на тему цветовой модели RGB.

📋 Пример 1: Базовый цвет

Задание: Какой цвет в цветовой модели RGB кодируется как RGB(255, 0, 0)?
Решение:
  1. R = 255 — красный на максимуме
  2. G = 0 — зелёный отсутствует
  3. B = 0 — синий отсутствует
  4. Вывод: только красный компонент активен
Ответ: Красный

📋 Пример 2: Смешанный цвет

Задание: Какой цвет в цветовой модели RGB кодируется как RGB(255, 255, 0)?
Решение:
  1. R = 255 — красный на максимуме
  2. G = 255 — зелёный на максимуме
  3. B = 0 — синий отсутствует
  4. Красный + Зелёный = Жёлтый
Ответ: Жёлтый

📋 Пример 3: Все компоненты равны

Задание: Какой цвет в цветовой модели RGB кодируется как RGB(128, 128, 128)?
Решение:
  1. R = G = B = 128
  2. Все компоненты равны — это оттенок серого
  3. Значение 128 — середина диапазона (0-255)
Ответ: Серый

📋 Пример 4: Определение по описанию

Задание: Укажите RGB-код для цвета "Голубой (Циан)".
Решение:
  1. Голубой (Циан) = Зелёный + Синий
  2. R = 0 (красный не участвует)
  3. G = 255 (зелёный на максимуме)
  4. B = 255 (синий на максимуме)
Ответ: RGB(0, 255, 255)

📋 Пример 5: Тёмный оттенок

Задание: Какой цвет в цветовой модели RGB кодируется как RGB(0, 128, 0)?
Решение:
  1. R = 0 — красный отсутствует
  2. G = 128 — зелёный на половине интенсивности
  3. B = 0 — синий отсутствует
  4. Это зелёный, но не максимальной яркости
Ответ: Тёмно-зелёный

💡 Советы для решения

✅ Алгоритм быстрого определения цвета

  • Шаг 1: Проверьте: все ли значения равны? → Серый
  • Шаг 2: Проверьте: все ли 0? → Чёрный
  • Шаг 3: Проверьте: все ли 255? → Белый
  • Шаг 4: Найдите ненулевые компоненты и примените правила смешения
  • Шаг 5: Учитывайте интенсивность (низкие значения = тёмный оттенок)

⚠️ Частые ошибки

  • Путают жёлтый (R+G) с оранжевым — оранжевый это R=255, G=128, B=0
  • Забывают, что серый получается при R = G = B
  • Не учитывают, что низкие значения дают тёмные оттенки
  • Путают аддитивную модель RGB с субтрактивной (CMYK)

Используйте интерактивный миксер для экспериментов с цветами!
