# ВПР - Интерактивные задания по информатике

Интерактивное веб-приложение для подготовки к Всероссийской проверочной работе (ВПР) по информатике для **7** и **8** классов. Проект содержит задания с различными типами интерактива, поддерживающими как мышь, так и сенсорные экраны.

Главная страница: **[vpr.html](vpr.html)** — выбор класса.

---

## 📘 7 класс (12 заданий)

Страница класса: [`vpr7/vpr7.html`](vpr7/vpr7.html)

### Режимы

| Режим                      | Описание                                                |
| -------------------------- | ------------------------------------------------------- |
| 🎯 **Режим экзамена**      | Пройти все 12 заданий подряд и получить итоговую оценку |
| 📋 **Генератор вариантов** | Создать и скачать варианты ВПР для печати               |

### Задания

| №   | Тема                               | Описание                                                                                                                |
| --- | ---------------------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| 1   | **Устройства компьютера**          | Распределение устройств по категориям: ввод, вывод, хранение и обработка информации                                     |
| 2   | **Пути к файлам**                  | Определение полного пути к папке или файлу после навигации по файловой системе                                          |
| 3   | **Типы файлов**                    | Распределение расширений файлов по категориям: текстовые, презентации, таблицы, графические, аудио, видео, архивы и др. |
| 4   | **URL-адреса**                     | Составление полного URL-адреса файла по заданным параметрам: протокол, сервер, путь и имя файла                         |
| 5   | **Логические задачи**              | Решение логических задач на сопоставление имён и фамилий на основе условий                                              |
| 6   | **Декодирование сообщений**        | Расшифровка сообщения с использованием таблицы кодов. Декодирование может быть однозначным с начала или с конца         |
| 7   | **Объём информации**               | Решение задач на вычисление объёма информации: сложение, вычитание, сравнение и перевод единиц измерения                |
| 8   | **Объём текстовой информации**     | Определение информационного объёма сообщения при кодировании символов с использованием алфавита заданной мощности       |
| 9   | **Передача информации**            | Решение задач на передачу информации: вычисление скорости, времени или объёма по двум известным величинам               |
| 10  | **Объём информации в предложении** | Определение информационного объёма предложения в байтах при кодировании Unicode                                         |
| 11  | **Цветовая модель RGB**            | Определение цвета по значениям RGB-компонентов. Интерактивный цветовой миксер в подсказке                               |
| 12  | **Форматирование текста**          | Определение элементов форматирования абзаца по его виду в текстовом редакторе: отступы, выступы, выравнивание           |

К каждому заданию прилагается страница с теорией (например, `vpr7-1-t.html`).

## 📗 8 класс (10 заданий)

Страница класса: [`vpr8/vpr8.html`](vpr8/vpr8.html)

### Задания

| №   | Тема                               | Описание                                                                                                                                                   |
| --- | ---------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1   | **Системы счисления**              | Перевод чисел между двоичной, восьмеричной, десятичной и шестнадцатеричной системами счисления                                                             |
| 2   | **Сравнение чисел**                | Сравнение целых чисел в различных позиционных системах счисления, нахождение числа в заданном диапазоне                                                    |
| 3   | **Сложение в системах счисления**  | Сложение чисел столбиком в позиционных системах счисления с основанием от 2 до 16, кроме десятичной                                                        |
| 4   | **Вычитание в системах счисления** | Вычитание чисел столбиком в позиционных системах счисления с основанием от 2 до 16, кроме десятичной                                                       |
| 5   | **Логические высказывания**        | Определение истинности составных высказываний, логические операции: И, ИЛИ, НЕ, построение таблиц истинности                                               |
| 6   | **Логические выражения**           | Правила записи логических выражений. Построение таблиц истинности логических выражений: И, ИЛИ, НЕ                                                         |
| 7   | **Алгоритм для исполнителя**       | Составление алгоритма преобразования числа с помощью команд исполнителя: сложение, вычитание, умножение, деление, возведение в квадрат, приписывание цифры |
| 8   | **Исполнитель Чертёжник**          | Анализ алгоритма для исполнителя Чертёжник, перемещающего перо по координатной плоскости: команды смещения и цикла                                         |
| 9   | **Анализ алгоритмов**              | Анализ программ на языке Pascal с условным оператором: логические операции and, or, not и скобки. Выбор пар чисел, для которых программа выведет "NO"      |
| 10  | **Таблицы истинности**             | Заполнение таблицы истинности для логического выражения с 3 переменными и 5 операциями: НЕ, И, ИЛИ. Использование скобок                                   |

К каждому заданию прилагается страница с теорией (например, `vpr8-1-t.html`).

## 🛠️ Технологии

- **HTML5** — структура страниц
- **CSS3** — стилизация, градиенты, анимации, адаптивный дизайн
- **JavaScript** — интерактивность, drag-and-drop API, touch events, генерация вариантов
- **Текстовые файлы данных** — разделение контента и логики

## 🚀 Запуск

Просто откройте `vpr.html` в браузере или запустите локальный сервер:

```bash
# С помощью Python
python -m http.server 8000

# С помощью Node.js
npx serve
```

Затем откройте http://localhost:8000 в браузере.

## ✨ Особенности

- **Адаптивный дизайн** — работает на десктопе и мобильных устройствах
- **Touch-поддержка** — перетаскивание пальцем на сенсорных экранах
- **Мгновенная проверка** — визуальная обратная связь при размещении элементов
- **Случайная генерация** — новые задания при каждом запуске
- **Подсказки** — помощь при затруднениях
- **Подробные результаты** — разбор ошибок после проверки
- **Режим экзамена** — прохождение всех заданий подряд с итоговой оценкой
- **Генератор вариантов** — создание и печать индивидуальных вариантов
- **Теоретические материалы** — страницы с теорией к каждому заданию
- **Яндекс.Метрика** — встроенная статистика посещений

## 💾 Формат данных

Данные хранятся в текстовых файлах с простой разметкой:

```
@Название категории
Элемент 1
Элемент 2
Элемент 3
```

Для добавления новых элементов просто отредактируйте соответствующий `.txt` файл в директории `vpr7/data/`.
