# ВПР - Интерактивные задания по информатике

Интерактивное веб-приложение для подготовки к Всероссийской проверочной работе (ВПР) по информатике. Проект содержит 11 заданий с различными типами интерактива, поддерживающими как мышь, так и сенсорные экраны.

## 🎯 Задания

### Задание 1: Устройства компьютера

Распределение устройств по категориям: ввод, вывод, хранение и обработка информации.

### Задание 2: Пути к файлам

Определение полного пути к папке или файлу после навигации по файловой системе.

### Задание 3: Типы файлов

Распределение расширений файлов по категориям: текстовые, презентации, таблицы, графические, аудио, видео, архивы и др.

### Задание 4: URL-адреса

Составление полного URL-адреса файла по заданным параметрам: протокол, сервер, путь и имя файла.

### Задание 5: Логические задачи

Решение логических задач на сопоставление имён и фамилий на основе условий.

### Задание 6: Декодирование сообщений

Расшифровка сообщения с использованием таблицы кодов. Декодирование может быть однозначным с начала или с конца.

### Задание 7: Объём информации

Решение задач на вычисление объёма информации: сложение, вычитание, сравнение и перевод единиц измерения.

### Задание 8: Объём текстовой информации

Определение информационного объёма сообщения при кодировании символов с использованием алфавита заданной мощности.

### Задание 9: Передача информации

Решение задач на передачу информации: вычисление скорости, времени или объёма по двум известным величинам.

### Задание 10: Объём информации в предложении

Определение информационного объёма предложения в байтах при кодировании Unicode.

### Задание 11: Цветовая модель RGB

Определение цвета по значениям RGB-компонентов. Интерактивный цветовой миксер в подсказке.

## 🛠️ Технологии

- **HTML5** — структура страниц
- **CSS3** — стилизация, градиенты, анимации, адаптивный дизайн
- **JavaScript** — интерактивность, drag-and-drop API, touch events
- **Данные в текстовых файлах** — разделение контента и логики

## 🚀 Запуск

Просто откройте `vpr7.html` в браузере или запустите локальный сервер:

```bash
# С помощью Python
python -m http.server 8000

# С помощью Node.js
npx serve
```

Затем откройте http://localhost:8000 в браузере.

## ✨ Особенности

- **Адаптивный дизайн** — работает на десктопе и мобильных устройствах
- **Touch-поддержка** — перетаскивание пальцем на сенсорных экранах
- **Мгновенная проверка** — визуальная обратная связь при размещении элементов
- **Случайная генерация** — новые задания при каждом запуске
- **Подсказки** — помощь при затруднениях с анимацией раскрытия
- **Подробные результаты** — разбор ошибок после проверки

## 📝 Формат данных

Данные хранятся в текстовых файлах с простой разметкой:

```
@Название категории
Элемент 1
Элемент 2
Элемент 3
```

Для добавления новых элементов просто отредактируйте соответствующий .txt файл.
